1. 格納物
　・ attributes.json（各馬の因子情報，血統情報）
　・ default_pedigree.json（各馬の祖先馬情報）
　・ sire.csv（牡馬の全血統データ）
　・ dam.csv（牝馬の全血統データ）
　・ Json用attrib自動生成.csv / xlsx
　　（牡牝の全血統データから因子情報・祖先馬情報を抜き出したもの）
　・csvTojson.py（自動生成.csvを，「書式を整えて」jsonにする）

2. 必要環境
　・Windows
　・Microsoft Excel
　・Python 3.系

3. 専門用語
　・幅優先記法 / 深さ優先記法：
　　前者は「父→母→父父→...」とする血統記載法。クロス判定処理に向く。
　　後者は「父→父父→父父父→...」とする血統記載法。htmlでのテーブル表示に向く。

4. 更新手順
　a. 更新馬の全血統を調べ，sire.csvかdam.csvに深さ優先記法で記載
　   Netkeibaはスクレイピング対策が厳しいため，jbisを推奨。
　b. JSON用attrib自動生成.xlsxを開く
  c. 因名が追加された場合，因名と因子情報を因名シートに記載
     因子情報のIDは，最適Mastersに準ずる。
　d. 新たに追加された牡馬データ（その祖先馬含む）の血統データを"祖先馬"シートに追加。
　　 配列は左から，馬名(半角＋因名），第１因子，第２因子，第３因子，子血統。
　　 子血統のIDは最適Mastersに順ずる。
　　 また，因名は一度テキストなどに書いてから貼り付ける。
     (ExcelのPhonetic関数で半角→全角に戻す処理をしているため。)
  e. 馬シートを全選択→削除し，sire.csvのデータとdam.csvのデータを上下に並べて貼り付ける。
  f. xlsxファイルを上書き保存する。
  g. Json用Attrib自動生成シートを選択し，このシートをcsvとして保存する。
  h. pythonでcsvTojsonを呼び出し実行する。
  i. json２つが生成される。
